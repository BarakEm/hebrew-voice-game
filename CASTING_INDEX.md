## Common Questions

### Can I cast the app to TV right now?
**Yes!** Open the web app in Chrome and use "Cast tab" (tap â‹® â†’ Cast). No code changes needed.

### Which method has the best quality?
**HDMI cable** - zero latency, perfect quality. Cost: $15-50 for adapter.

### Which method is easiest?
**Chrome Cast Tab** - works in 30 seconds if you have Chromecast/Google TV.

### Do I need to write code?
**No!** Methods 1, 3, 4, 5, 6 work without coding. Only Method 2 (Google Cast API) requires code changes.

### Will audio work?
**Yes!** In wireless methods, microphone stays on your phone. In HDMI, audio goes through cable.

### Can multiple people cast to the same TV?
**Yes!** With Chromecast, anyone can request to cast. Current user must stop casting first.
